# Student Result Management System

A beginner-level C++ project for managing student marks and results using arrays, loops, and conditional statements.

## Author

Farzana Akter

Department of CSE

Southeast University

## Features

* Add student marks
* Display all student marks
* Find highest mark
* Calculate average marks
* Check pass or fail status
* Menu-driven system

## Concepts Used

* Arrays
* For Loops
* If-Else Statements
* Nested Conditions
* Basic Calculations
* Menu-Based Programming

## Purpose

This project was created to practice fundamental C++ programming concepts and improve problem-solving skills through a simple result management system.
